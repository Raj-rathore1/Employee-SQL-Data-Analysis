# Employee-SQL-Data-Analysis


## Project Overview
This project focuses on analyzing employee data using SQL.  
The dataset includes employee details, department information, salaries, and attendance records.

The objective of this project is to perform data cleaning, generate KPIs, and extract meaningful insights using SQL queries.

---

## Dataset Information

The project uses the following datasets:

- Employees
- Departments
- Salaries
- Attendance

These datasets are stored inside the `datasets/` folder.

---

## Tools & Technologies Used

- SQL (MySQL)

---

## Key SQL Concepts Used

- SELECT Statements
- INSERT & DELETE Operations
- Aggregate Functions
- GROUP BY 
- JOIN Operations
- Common Table Expressions (CTE)
- Stored Procedures
- KPI Analysis
- Data Cleaning Techniques
